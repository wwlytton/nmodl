template<unsigned int SIZE = 4>
class nmodl::utils::SingletonRandomString< SIZE >

Singleton class for random strings.

Singleton class for random strings that are appended to the Eigen matrices names that are used in the solutions of nmodl::visitor::SympySolverVisitor and need to be the same to be printed by the nmodl::codegen::CodegenCVisitor
