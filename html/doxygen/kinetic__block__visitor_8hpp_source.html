32 /**
33  * \class KineticBlockVisitor
34  * \brief %Visitor for kinetic block statements
35  *
36  * Replaces each KINETIC block with a DERIVATIVE block
37  * containing a system of ODEs that is equivalent to
38  * the original set of reaction statements
39  *
40  * Note: assumes that the order of statements between the first and last
41  * reaction statement (those starting with "~") does not matter.
42  *
43  * If there is a CONSERVE statement it is rewritten in an equivalent
44  * form which is then be used by the SympySolver visitor to replace
45  * the ODE for the last state variable on the LHS of the CONSERVE statement
46  *
47  */
