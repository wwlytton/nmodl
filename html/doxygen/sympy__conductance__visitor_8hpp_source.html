37 /**
38  * \class SympyConductanceVisitor
39  * \brief %Visitor for generating CONDUCTANCE statements for ions
40  *
41  * This class visits each ion expression `I = ...` in the `BREAKPOINT`
42  * and symbolically differentiates it to get `dI/dV`, i.e. the conductance.
43  * If this coincides with an existing global variable `g` (e.g. when
44  * `I` is linear in `V`) then it will add
45  *
46  * \code{.mod}
47  * CONDUCTANCE g USEION I
48  * \endcode
49  *
50  * If `dI/dV` is a more complicated expression, it generates a new unique
51  * variable name `g_unique`, and adds two lines
52  *
53  * \code{.mod}
54  * CONDUCTANCE g_unique USEION I
55  * g_unique = [dI/dV expression]
56  * \endcode
57  *
58  * If an ion channel already has a `CONDUCTANCE` statement then it does
59  * not modify it.
60  */
